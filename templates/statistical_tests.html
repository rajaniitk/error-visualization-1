📊 Statistical Tests Dashboard

Perform comprehensive statistical analysis and hypothesis testing on your data

📈 Descriptive Statistics

📋 Requirements:

  • ✅ Any Data Type Works with both numeric and categorical columns
  • 📊 Multiple Columns Select one or more columns for analysis
  • 📈 Summary Stats Generates mean, median, mode, standard deviation, etc.
Hold Ctrl/Cmd to select multiple columns

📊 Normality Tests

📋 Requirements:

  • 📊 Numeric Data Requires one numeric column
  • 📈 Sample Size Minimum 3 data points (varies by test)
  • 🎯 Shapiro-Wilk 3-5000 samples
  • 🎯 Anderson-Darling 5+ samples
  • 🎯 Jarque-Bera 20+ samples

🔗 Correlation Tests

📋 Requirements:

  • 📊 Numeric Data Requires two numeric columns
  • 📈 Sample Size Minimum 3 paired observations
  • 🎯 Pearson Linear relationship, normal distribution
  • 🎯 Spearman Monotonic relationship
  • 🎯 Kendall Ordinal data, robust to outliers

🧪 T-Tests

📋 Requirements:

  • 📊 Numeric Data Requires numeric dependent variable
  • 📈 One-Sample 1 numeric column, test value, 3+ observations
  • 📊 Two-Sample 1 numeric + 1 categorical (2 groups), 4+ observations
  • 🔗 Paired 2 numeric columns, 3+ paired observations
  • 🎯 Assumptions Normal distribution, equal variances (for independent)

📊 Analysis of Variance (ANOVA)

📋 Requirements:

  • 📊 Numeric Data Requires numeric dependent variable
  • 📈 One-Way 1 categorical independent variable (2+ groups)
  • 📊 Two-Way 2 categorical independent variables
  • 🎯 Sample Size Minimum 5 observations total, 2+ per group
  • 🎯 Assumptions Normal distribution, equal variances
Two-Way ANOVA: Please select a second independent variable for two-way analysis.

🎯 Chi-Square Tests

📋 Requirements:

  • 📊 Categorical Data Requires categorical variables
  • 📈 Independence 2 categorical variables
  • 🎯 Goodness of Fit 1 categorical variable
  • 📊 Sample Size Minimum 5 expected frequency per cell
  • 🎯 Assumptions Independent observations, adequate sample size

📊 Non-Parametric Tests

📋 Requirements:

  • 📊 Ordinal Data Works with ordinal or numeric data
  • 📈 Mann-Whitney 2 groups, 3+ observations per group
  • 🔗 Wilcoxon 2 paired columns, 6+ observations
  • 🎯 Kruskal-Wallis 2+ groups, ordinal/numeric data
  • 📊 Friedman 3+ columns, 6+ complete observations
Hold Ctrl/Cmd to select multiple columns (minimum 3)

📊 Variance Tests

📋 Requirements:

  • 📊 Numeric Data Requires 2+ numeric columns
  • 📈 Levene's Test Robust to non-normality
  • 🎯 Bartlett's Test Assumes normal distribution
  • 📊 Fligner-Killeen Non-parametric alternative
  • 🎯 Sample Size Minimum 3 observations per group
Hold Ctrl/Cmd to select multiple columns (minimum 2)

🎯 McNemar Test

📋 Requirements:

  • 📊 Categorical Data Requires 2 categorical columns
  • 📈 2x2 Table Creates 2x2 contingency table
  • 🔗 Paired Data For paired/matched samples
  • 🎯 Sample Size Adequate discordant pairs needed
  • 📊 Usage Before/after comparisons

📊 Multiple Comparisons

📋 Requirements:

  • 📊 Numeric Data Requires numeric dependent variable
  • 📈 Post-hoc Used after significant ANOVA
  • 🎯 Tukey HSD Equal sample sizes, conservative
  • 📊 Bonferroni Very conservative, any sample sizes
  • 🎯 Holm-Bonferroni Less conservative than Bonferroni
